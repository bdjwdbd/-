# 模块归属分析文档目录

这个目录存放所有新模块的归属分析文档。

## 文档命名规范

```
YYYY-MM-DD-模块名.md
```

例如：`2026-04-16-statamanager.md`

## 使用方法

### 1. 生成分析文档

```bash
node scripts/generate-module-analysis.js
```

### 2. 填写文档

打开生成的文档，填写以下内容：
- 分析者姓名
- 审批状态
- 其他备注

### 3. 提交审批

将文档提交给架构师审批。

### 4. 开始开发

审批通过后，按照文档中的计划开始开发。
