### 🎯 Lane 优先级模型

```
┌─────────────────────────────────────────────────────────────────┐
│                      Lane 优先级模型                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  31 位位掩码设计：                                                │
│  ─────────────────────────────────────────────────────────────  │
│                                                                  │
│  Sync        0b0000000000000000000000000000001  (最高优先级)     │
│  UserInput   0b0000000000000000000000000010000                  │
│  Default     0b0000000000000000000000000100000                  │
│  Transition  0b0000000001111111111111111000000  (16 个过渡 Lane) │
│  Retry       0b0000111110000000000000000000000  (5 个重试 Lane)  │
│  Idle        0b0100000000000000000000000000000  (最低优先级)     │
│                                                                  │
│  核心操作：                                                       │
│  • getHighestPriorityLane(lanes) - O(1) 获取最高优先级          │
│  • mergeLanes(a, b) - 合并优先级                                 │
│  • includesLane(lanes, lane) - 检查是否包含                      │
│                                                                  │
│  借鉴来源：React Lane 模型                                        │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### 🌳 B+ 树知识索引

```
┌─────────────────────────────────────────────────────────────────┐
│                    B+ 树知识索引                                  │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  结构特点：                                                       │
│  ─────────────────────────────────────────────────────────────  │
│  • 所有值存储在叶节点                                            │
│  • 叶节点通过链表连接，支持高效范围查询                          │
│  • 内部节点只存储键和子节点指针                                  │
│  • 默认阶数：64                                                  │
│                                                                  │
│  复杂度：                                                         │
│  ┌──────────────────────────────────────────────────────────┐   │
│  │ 精确查找：O(log n)                                        │   │
│  │ 范围查询：O(log n + k)，k 为结果数                        │   │
│  │ 插入：    O(log n)                                        │   │
│  │ 删除：    O(log n)                                        │   │
│  └──────────────────────────────────────────────────────────┘   │
│                                                                  │
│  支持的搜索：                                                     │
│  • 精确查找：find(key)                                           │
│  • 范围查询：range(start, end)                                   │
│  • 前缀搜索：searchPrefix(prefix)                                │
│  • 模糊搜索：searchFuzzy(query, maxDistance)                     │
│                                                                  │
│  借鉴来源：SQLite B+ tree 存储结构                                │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```
