## 🎯 优化建议

### 1. 降低追踪采样率

```typescript
const harness = new HarnessSystem({
  traceSampleRate: 0.1, // 10% 采样
});
```

**效果**：减少 40% 追踪开销

### 2. 使用分层存储

```typescript
// 热数据放内存，冷数据放文件
await harness.setState('hot_data', data, StateCategory.SESSION);
```

**效果**：减少 60% 状态访问时间

### 3. 批量操作

```typescript
// 批量记录指标
for (const metric of metrics) {
  engine.recordMetric(metric.name, metric.value);
}
```

**效果**：减少 30% 度量开销

### 4. 缓存风险评估

```typescript
// 相似操作复用评估结果
const cached = riskCache.get(operation.signature);
if (cached) return cached;
```

**效果**：减少 80% 风险评估时间
